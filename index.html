Challenge 1

    • 150px diameter

Challenge 2

  • Box containing rectangle:
    • 500px height
    • 2px dashed cornflowerblue border
  • Rectangle:
    • Height: 100% of the containing div
    • Width: 50% of the containing div
    • Background color: cornflowerblue
    • Border: 2px solid purple
  • Challenge:
    • Horizontally center the rectangle using margin.
    • Don't allow borders to overlap

Challenge 3

  • Yellow circle:
    • 150px diameter
    • Background color: yellow
    • Location: 10px from the bottom and 10px from the left
